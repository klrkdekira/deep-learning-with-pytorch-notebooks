# Deep Learning with Python — PyTorch Notebooks

PyTorch ports of the companion notebooks for [Deep Learning with Python, third edition (2025)](https://www.manning.com/books/deep-learning-with-python-third-edition) by François Chollet and Matthew Watson.

The original notebooks use [Keras 3](https://keras.io) and support JAX, TensorFlow, and PyTorch backends. This repo rewrites each chapter using native PyTorch — no Keras dependency.

> Original repository: [fchollet/deep-learning-with-python-notebooks](https://github.com/fchollet/deep-learning-with-python-notebooks)
